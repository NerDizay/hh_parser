# Парсер вакансий HH.ru на Selenium

Скрипт для авторизации и парсинга вакансий на сайте [hh.ru](https://hh.ru) с использованием Selenium, webdriver-manager и environs.

## Требования

- Python 3.8+
- Установленный браузер (Google Chrome рекомендуется)
- Переменные окружения для авторизации

## Установка

1. Клонируйте репозиторий или скачайте файлы.
2. Установите зависимости:
   ```bash
   pip install -r req.txt
   ```

3. Настройте переменные окружения:
   Создайте файл `.env` в корне проекта (он уже создан, но данные нужно заполнить):
   ```env
   HH_EMAIL=ваш_email@example.com
   HH_PASSWORD=ваш_пароль
   ```

## Запуск

Запустите скрипт командой:
```bash
python main.py
```

## Как это работает

1. Скрипт считывает логин и пароль из файла `.env` с помощью библиотеки `environs`.
2. Запускает браузер в режиме `headless` (без графического интерфейса).
3. Выполняет функцию `login`, которая переходит на страницу авторизации hh.ru.
4. После авторизации выполняет поиск вакансий (в примере — по запросу "Python") и выводит заголовки первых найденных вакансий.

## Примечание

- Убедитесь, что ваш аккаунт hh.ru активен.
- Сайт hh.ru может иметь защиту от ботов. Если скрипт не работает, возможно, потребуется добавить задержки (`time.sleep`) или изменить селекторы элементов, так как структура сайта может меняться.
- Не используйте скрипт для агрессивного парсинга, чтобы не нарушать правила использования сайта.
